# Installation

[![Get it from NuGet](https://raw.githubusercontent.com/hmlendea/readme-assets/master/badges/stores/nuget.png)](https://nuget.org/packages/NuciLog)

**.NET CLI**:
```bash
dotnet add package NuciLog
```

**Package Manager**:
```powershell
Install-Package NuciLog
```
